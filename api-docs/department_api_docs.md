# Department Management (Quản lý phòng ban) API Reference

This document details the APIs related to managing departments/divisions within an agency unit.

## Overview

*   **List Departments**: Code **079**
*   **Create/Update Department**: Code **077**
*   **Delete Department**: Code **080**

---

## 1. List Departments (Code 079)

Retrieves the list of departments/divisions within the unit.

*   **Purpose**: Get all departments for display or selection.
*   **Detailed Documentation**: [code-079.md](code-079.md)

---

## 2. Create/Update Department (Code 077)

Creates a new department or updates an existing one.

*   **Purpose**: Add or Modify a Department.
*   **Detailed Documentation**: [code-077.md](code-077.md)

---

## 3. Delete Department (Code 080)

Deletes an existing department.

*   **Purpose**: Remove a department.
*   **Detailed Documentation**: [code-080.md](code-080.md)
