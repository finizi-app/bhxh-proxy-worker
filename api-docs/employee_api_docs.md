# Employee Management (Quản lý lao động) API Reference

This document details the APIs related to managing employees within an agency unit.

## Overview

*   **Get Employee List**: Code **067**
*   **Get Employee Detail**: Code **172**

---

## 1. Get Employee List (Code 067)

Retrieves the list of employees for the current unit.

*   **Purpose**: "Quản lý lao động".
*   **Detailed Documentation**: [code-067.md](code-067.md)

---

## 2. Get Employee Detail (Code 172)

Attempts to retrieve details for a specific employee ID.

*   **Purpose**: Get detailed info for an employee.
*   **Detailed Documentation**: [code-172.md](code-172.md)

---

## 3. Upload Employee List (Code 112)

Uploads an Excel file to import employee data.

*   **Purpose**: Bulk import of employees.
*   **Detailed Documentation**: [code-112.md](code-112.md)
